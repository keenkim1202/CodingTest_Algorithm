# CodingTest_Algorithm
코딩테스트 대비와 알고리즘 연습을 위한 레포입니다.


# 작성 방법
- 모든 문제는 swift로 풀이합니다.
- 관련 주제에 해당하는 폴더에 풀이한 문제의 `swift 파일`과 개념에 대한 `내용정리 파일`을 업로드합니다.
- 풀이한 문제 파일을 업로드할 때는 `문제링크`, `문제 요약`, `풀이`를 포함하여 작성합니다.
- '문제 및 파일 링크'에는 `문제 출제 플랫폼`, `문제 제목`, `풀이`, `난이도`를 작성합니다.
- 난이도는 상(🔴),중(🟠 ),하(🟢)로 표기합니다.
